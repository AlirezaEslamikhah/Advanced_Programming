The computer machine was first invented in 1837 by Charles Babbage.
This machine was able to calculate multi-sentence functions.
Actually it was as modern as today's computers because it contained cpu and storage.

It was relatively cheaper than today's ones,which only cost 750 dollars.
The main revolution was in 1981 where IBM company introduced  the first modern personal computer which included mouse and screen.
It got famous rapidly and many applications were made for it one year later.
